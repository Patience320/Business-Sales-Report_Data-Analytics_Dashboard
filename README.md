# Business Sales Report (Interactive Dashboard creation using PowerBI)
## Business Sales Report Dashboard  
This project analyzes sales performance for a retail superstore across categories, customer segments, regions, and shipping modes. The dashboard highlights key insights such as revenue trends, customer distribution, product category contributions, and logistics efficiency.

## Key Features:

- Sales and profit analysis by year, quarter, and category

- Customer segmentation (Consumer, Corporate, Home Office)

- Regional and state-level performance breakdown

- Shipping mode and cost analysis

- Sub-category insights (Phones, Copiers, Chairs, etc.)

## Process 
- Verified data for any missing values and anomalies.
- Standardized data types across fields (converted relevant columns to text).
- Promoted first row to headers for better structure on some fields.
- Removed Postal Code field due to high number of null/empty values.
- Converted Order Date from text to date format.
- Identified and removed duplicate records.
